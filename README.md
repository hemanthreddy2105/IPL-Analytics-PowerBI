# 🏏 IPL Analytics Dashboard

## 📌 Project Overview

This project is an end-to-end IPL Analytics Dashboard developed using PostgreSQL and Power BI. It provides interactive insights into team performance, batting statistics, bowling performance, venue analysis, and overall IPL trends.

---

## 🚀 Tools Used

- PostgreSQL
- SQL
- Power BI

---

## 📊 Dashboard Pages

### Executive Dashboard
- Total Matches
- Total Runs
- Total Players
- Total Teams
- Total Wickets
- Total Sixes
- Total Fours

### Batting Dashboard
- Top Run Scorers
- Strike Rate Analysis
- Most Sixes
- Most Fours
- Runs by Season

### Bowling Dashboard
- Top Wicket Takers
- Economy Rate
- Dot Balls
- Wickets by Season

### Team Dashboard
- Matches Won
- Team Performance
- Toss Analysis
- Team-wise Statistics

### Venue Dashboard
- Matches by Venue
- Runs by Venue
- Wickets by Venue
- Toss Decision Analysis
